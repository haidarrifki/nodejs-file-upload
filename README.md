# Node.js File Upload

This project is simple File Upload application built in Node.js using Express.js and Multer.

## Pre-requisite
1. Install Node.js >= v12.x.

## Next steps
- clone this repository
- ```yarn install```
- ```yarn dev```
- open browser `http://localhost:1337`
